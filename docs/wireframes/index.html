SIFTQ UI Wireframes

Matrix MVP の画面状態と遷移関係

Current combined wire
1. Empty 空 area は中央の + から作成を開始する。 2. Create 対象 area で title を入力する。 3. Cards 作成済み card、count、Edit 入口。 4. Editing title update の保存 / キャンセル。 5. Dragging matrix area 間、または terminal area へ移動。 6. Terminal Drop Done / Skipped drop 後は matrix から非表示。

状態別ワイヤー

状態 確認したいこと 次の遷移
Empty No cards を出さず、空 area に + を置く。 + click → Create
Create 入力欄、disabled、error、作成位置。 Create success → Cards
Cards card 表示、count、Edit 入口、長い title。 Edit click → Editing / drag start → Dragging
Editing title update の入力、保存、キャンセル、validation。 Save / Cancel → Cards
Dragging drag 中、drop target、viewport 内操作範囲。 Drop on matrix → Cards / Drop on terminal → Terminal Drop
Terminal Drop Done / Skipped に count を出さず、drop 後 card を matrix から消す。 次操作待ち。v1 では復元 UI なし。
