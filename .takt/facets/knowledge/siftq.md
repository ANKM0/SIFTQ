# SIFTQ Knowledge

SIFTQ is a local-first task matrix application. The current MVP is a browser
SPA built with React, TypeScript, Vite, and dnd-kit. Later local application
work targets Rust and Tauri.

Ticket-driven AI development is repository workflow tooling. It must remain
outside the product runtime and should respect existing Taskfile, CoDD, branch,
and commit conventions.
